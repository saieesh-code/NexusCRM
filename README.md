# NexusCRM 🚀

A modern full-stack CRM (Customer Relationship Management) platform designed for lead management, pipeline tracking, analytics, and sales workflow automation.

## 🌐 Live Demo

Frontend: https://nexus-crm-two-bice.vercel.app/

Backend API: https://nexus-crm-backend-ol29.onrender.com/

---

# ✨ Features

## 📊 Dashboard Analytics
- Real-time CRM statistics
- Lead conversion tracking
- Pipeline overview
- Monthly lead charts
- Activity timeline

## 👥 Lead Management
- Add / Edit / Delete leads
- Search and filter leads
- Status management
- Source tracking
- Assigned sales representatives

## 📌 Pipeline Board
- Interactive Kanban workflow
- Drag-and-view sales stages
- Organized lead progression
- Clean enterprise UI

## 📝 Notes & Activities
- Add notes to leads
- Activity logging
- Timeline history
- Follow-up tracking

## 📈 Analytics
- Funnel analytics
- Source-based insights
- Monthly trends
- Conversion metrics

## 📤 Export Functionality
- CSV export support
- Structured lead data download

## 🎨 Modern UI
- Responsive design
- Professional dashboard layout
- Enterprise CRM aesthetics
- Interactive charts with Chart.js

---

# 🛠️ Tech Stack

## Frontend
- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Deployment
- Frontend → Vercel
- Backend → Render
- Database → MongoDB Atlas
